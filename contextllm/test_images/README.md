# 📁 테스트 이미지 폴더

이 폴더에 테스트할 이미지를 넣어주세요.

## 사용 방법

1. **이미지 준비**
   - 스크린샷, 사진, 또는 다른 이미지를 이 폴더에 복사
   - 지원 형식: `.jpg`, `.jpeg`, `.png`, `.bmp`, `.gif`

2. **테스트 실행**
   ```bash
   python tests/test_custom_image.py
   ```

3. **이미지 선택**
   - 프로그램이 이 폴더의 이미지 목록을 표시
   - 번호로 선택하거나 다른 경로 입력 가능

## 예시 테스트 시나리오

### 시나리오 1: 화재 감지 테스트
- 이미지: 불/연기 사진
- 음성 텍스트: "불이야!"
- 예상 결과: CRITICAL, 긴급 상황 감지

### 시나리오 2: 일상 대화 테스트
- 이미지: 컴퓨터 화면 스크린샷
- 음성 텍스트: "지금 코딩 중이야"
- 예상 결과: LOW, 일상 상황

### 시나리오 3: 음성-시각 불일치 테스트
- 이미지: 게임 화면
- 음성 텍스트: "도와주세요!"
- 예상 결과: 불일치 감지, 낮은 긴급도

## 테스트 결과

모든 테스트 결과는 `data/logs/` 폴더에 JSON 형식으로 저장됩니다.
